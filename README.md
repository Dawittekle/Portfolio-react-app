## 🚀 Getting Started

1. Clone the repo:

   ```bash
   git clone https://github.com/Dawittekle/Portfolio-react-app.git
   cd your-repo-name
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm start
   ```

> This project uses Create React App (CRA).
